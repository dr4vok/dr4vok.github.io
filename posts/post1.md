**What is the "Operating System" ("OS" for short) ?**

It’s the parts of the computer software that is responsible for basic use and administration this includes

- Boot loader
- Kernel
- device drivers
- system utilities
- command shell or other UI

in other words the stuff that you use everyday without you giving it much of attention !

**What is the term "system" refers to?**

- The term "system" refers to the OS and all applications running on top of it to achieve a variety of tasks

**What is the "bootloader"?**

- Its a program that is responsible for booting the OS and what it first do is that it loads the OS (specifically the kernel) into the Random Access Memory (for more info check the resources down below!)

**What is the "kernel"**

- Its the core part of the computer’s OS software; it manages the software that provides services for all other parts of the system, It also manages the hardware and distributes system resources

think of it like the manager that manages the whole company

**What is the "device driver"?**

- Its a software that controls a particular type of device that is attached to the computer.
    It provides a software access to the hardware devices without needing to know the boring details of the hardware

**What is the "system utilities"?**

- Its a program that performs specific tasks to enhance productivity, efficiency, functionality, or maintenance of a computer system

**What is the "command shell" and "GUI" and what is the difference?**

- The command shell is a program that let you interact with your computer though it. think of it like "CMD" on windows and "BASH" on linux
- The GUI (Graphical User Interface) is what you are dealing with everyday from interactive , visually appealing applications
